# Deskdrink

*Just a small python application for windows, to remind you to drink something.*

## Features

* Set a personalized time for the interval of the reminders

* Set personalized title, message, icon and audio(*icon can't be set on MacOS*)

* Get a system notification via notify-py (*uses Libnotify on Linux*)

### Planned

* Start with system bootup

* Version for macos and linux

## Installation

### Windows

Check, if you're using an x64/x86 or arm-based machine.

Download the newest release for your machine and run deskdrink-installer.exe, deskdrink-installer-x86.exe, or deskdrink-installer-arm64.exe to start the installation.

## How to use

Start the application manually. It runs in the background and notifies you, when the start was successful. You should have a new system tray icon, which you can use to change the settings (it opens a window, all changes or resets must be saved) or quit the application.
